Code Repository

uPortal uses CVS for its code repository. You can access the uPortal code via anonymous access using a CVS client. Most *nix operating systems have a CVS client available and there are CVS clients available for Microsoft Windows (WinCVS being a popular choice).

The JA-SIG CVS Repository is a web interface to the uPortal CVS code repository. You can use this to browse through all the branches of the uPortal Code.

If you are interested in contributing to the uPortal coding effort, refer to the Getting Involved section of the website for more information.

Anonymous access to CVS:

The uPortal project's CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. When prompted for a password for anonymous, simply press the Enter key.

cvs -d:pserver:anonymous@mis105.mis.udel.edu:/home/cvs/jasig login

cvs -d:pserver:anonymous@mis105.mis.udel.edu:/home/cvs/jasig co portal

Updates from within the module's directory do not need the -d parameter.
